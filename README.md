反証条件（先に定義・変更不可）
ZCLPは以下の条件が出た場合、棄却または修正とする。
① log-k空間の周期振動
SDSS/DESIで Δ(ln ℓ) ≈ 4 の周期が検出されない
検出されてもΛCDMで同等説明可能
複数データセットで周期値が不一致
② スケール階層比
実測スケール比がα（≈55）から2σ以上ずれる
階層ピークがランダム分布と統計的に区別できない
③ w(z)曲線
DESIでΛCDMと有意差が出ない
Ωの値がΩ_spaceにもΩ_massにも一致しない
高zで振幅発散、ΛCDMより明らかに悪いフィット
